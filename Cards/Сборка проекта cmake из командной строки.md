# Сборка проекта cmake из командной строки

``` shell
# Перейдите в директорию с вашим проектом
cd /path/to/your/project

# Создайте директорию для сборки (обычно это "build")
mkdir build
cd build

# Запустите CMake для генерации файлов сборки
cmake ..

# Запустите сборку
cmake --build .

```
